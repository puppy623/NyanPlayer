# NyanPlayer üéß

**A simple and lightweight music player for Android.**

NyanPlayer is designed to be a fast, easy-to-use music player that works entirely **offline** and in the background. It was originally created as a learning project in Java and Kotlin, but has evolved into a fully functional and pleasant music experience.


---

## ‚ú® Features

NyanPlayer focuses on core functionality and a straightforward user experience:

* **Lightweight & Efficient:** Designed for minimal resource usage.
* **Offline Playback:** Enjoy your music anywhere without needing an internet connection.
* **Background Play:** Keep the tunes going while you use other apps.
* **Playback Controls:**
    * **Loop/Repeat:** Easily loop the current track.
    * **Shuffle:** Randomize your listening experience.
    * Standard Play/Pause, Skip controls.
    * * **Playlists:** Create and manage your own custom song lists.
    * * **Customization:**
    * **Interchangeable Accent Colors:** Personalize the app's look and feel.
    * **Simple Top Offset Slider:** Adjusts the top layout to avoid interference from camera cutouts or notches.
